# VCI

これはVCI作成用のプロジェクトです。
UnityのEditorから `VCI` をエクスポートすることができます。

* Unity-2019.4
* .Net-4.5

をサポートしています。

このライブラリはアルファバージョンで機能の変更があります。
製品に組み込まないでください。Runtimeのエクスポートは未確認です。
